==========================================================================
                     XSP 支援 PCG 90 度回転ライブラリ
                         ver 1.03     by よっしん
==========================================================================




==========================================================================
                            本ライブラリの説明
==========================================================================

本ライブラリを用いることにより、16x16 ドット及び 8x8 ドットの PCG データを
右又は左に 90 度回転したように書き換えることができます。本ライブラリが対応
しているのは、X68K の PCG エリアと同じベタ形式フォーマットの PCG データで
す（XSP で使うフォーマットと同じです）。

XSP の縦画面モードを使用する場合の、PCG データの書き換え等にお勧めです。


==========================================================================
                               各関数の説明
==========================================================================

以下に各関数の使用法を説明します。


--------------------------------------------------------------------------

● 16x16 ドット PCG データの 90 度単位回転

書式：void pcg_roll90(void *pcg, int lr);

引数：void *pcg : 16x16 ドットの PCG データへのポインタ
      int lr    : 1 のとき右 90 度回転
                 -1 のとき左 90 度回転

戻り値：無し


★アセンブラ対応版

入力：A0.L = *pcg
      D0.l = lr

出力：無し

破壊レジスタ：無し


機能：指定されたアドレスから始まる 16x16 ドットの PCG データ 1 個を、指定
      方向に 90 度回転させます。



--------------------------------------------------------------------------

● 8x8 ドット BG データの 90 度単位回転

書式：void bgpcg_roll90(void *pcg, int lr);

引数：void *pcg : 8x8 ドットの PCG データへのポインタ
      int   lr  : 1 のとき右 90 度回転
                 -1 のとき左 90 度回転

戻り値：無し


★アセンブラ対応版について

入力：A0.L = *pcg
      D0.L = lr

出力：無し

破壊レジスタ：無し


機能：指定されたアドレスから始まる 8x8 ドットの PCG データ 1 個を、指定方
      向に 90 度回転させます。
